## 产品需求

安装"妙想模拟组合管理"技能（mx-moni），用于获取和管理用户持仓信息。根据用户提供的安装指南文档，需要完成以下任务：

1. **下载技能安装指南文档**：获取详细的安装说明
2. **验证系统环境**：确认Node.js版本符合要求（22+）
3. **安装mx-moni技能**：

- 清理可能存在的旧版本技能
- 下载技能压缩包并解压到正确的技能目录
- 验证技能安装是否成功

4. **配置永久环境变量**：设置MX_APIKEY环境变量，用于妙想API的身份验证

## 核心功能

- 安装mx-moni技能，扩展系统持仓管理能力
- 配置API密钥，获取妙想模拟组合的访问权限
- 完成技能的基础验证，确保能够正常调用API

## 技术方案

### 技术栈选择

- **Node.js**：技能运行所需环境（版本要求22+）
- **CodeBuddy Skills Framework**：技能安装框架
- **Zsh Shell**：macOS默认shell，用于配置环境变量

### 实施方法

1. **节点版本验证**：使用node -v验证Node.js版本是否满足要求
2. **文件操作**：使用shell命令进行文件下载、解压和目录操作
3. **技能安装**：将技能文件复制到CodeBuddy技能目录
4. **环境变量配置**：在~/.zshrc中添加export指令，然后重新加载配置

### 实施注意事项

- **安全性**：使用rm -rf命令清理旧版本时需要谨慎，确认目标目录
- **权限管理**：确保对~/.codebuddy/skills/目录有写权限
- **环境变量持久化**：在~/.zshrc中添加export而不是临时设置
- **技能验证**：通过检查技能目录和文件是否存在确认安装成功

### 系统架构

简单的单任务执行流程，无复杂架构设计：

1. 环境检查 → 2. 文件清理 → 3. 下载解压 → 4. 技能安装 → 5. 环境变量配置

### 目录结构

安装完成后将创建以下目录结构：

```
/Users/liuyi/.codebuddy/skills/mx-moni/
  ├── SKILL.md           # 技能描述文档
  ├── agent.py           # 主技能文件
  ├── package.json       # Node.js依赖配置（如果有）
  └── ...                # 其他技能文件
```

### 关键技术点

1. **技能目录确定**：基于现有mx-data、mx_search等技能的位置确定安装目录
2. **环境变量设置**：使用Zsh的配置文件~/.zshrc，通过export MX_APIKEY=...设置
3. **技能验证**：检查技能目录是否存在，确认文件数量合理
