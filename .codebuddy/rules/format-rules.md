# 项目规则

## 报告格式调整规则

### 飞书报告格式要求
1. **简洁优于详尽**：官方解读浓缩一句话放在指标后面，不单独开大块
2. **数据不能截断**：指标+趋势+解读需在同一行完整显示
3. **格式调整原则**：每次修改代码后必须发送飞书查看实际效果

### 指标展示格式
- **CPI/PPI**：极简格式 `CPI+1.3%↑/PPI-0.9% 春节因素影响，CPI环比涨幅创近两年最高`
  - 数据+趋势+解读紧凑呈现
  - 含关键数据点（环比方向、核心解读）
- **PMI**：方案C格式 `PMI 制造业49.0↓ / 非制造业49.5↑ 受春节因素影响，制造业回落但非制造业回暖`
  - 同时说明制造业和非制造业数据、趋势与解读

### 信号标识
- 🟢 买入/扩张
- 🟡 中性/临界
- 🔴 卖出/收缩

### 终端输出格式
- 结构清晰，分层缩进（├ / └ / │）
