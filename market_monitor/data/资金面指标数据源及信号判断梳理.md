# 资金面各指标数据源及信号判断梳理

> 生成时间：2026年3月24日
> 报告路径：`market_monitor/data/macro_report_YYYY-MM-DD.html`

## 一、核心资金观察指标

### 1. 指南针活跃市值

| 项目 | 说明 |
|------|------|
| **指标名称** | 指南针活跃市值 |
| **数据存储** | `market_monitor/data/znz_active_cap.csv` |
| **数据格式** | date(YYYY-MM-DD), active_cap(亿元), chg_pct(%), signal, source |
| **数据来源** | 手动上传（HTML报告界面支持手动录入） |
| **录入方式** | 点击"手动上传"链接，输入：活跃市值点数(亿元)、涨跌幅(%)、日期 |

#### 信号判断规则

| 涨跌幅条件 | 信号类型 | 信号描述 | 仓位建议 |
|------------|----------|----------|----------|
| 单日涨幅 ≥ +4% | `incremental` | 🟢 增量资金入场 | 40% |
| 单日跌幅 ≤ -2.3% | `exit` | 🔴 资金离场警示 | 0-10% |
| 其他情况 | `neutral` | 🟡 观望 | 20% |

#### 数据代码位置
- 信号计算：`market_monitor/data_sources/capital.py` → `_calc_znz_signal()`
- 仓位建议：`market_monitor/data_sources/capital.py` → `_suggest_position()`
- 数据读取：`market_monitor/data_sources/capital.py` → `fetch_znz_active_cap()`

## 二、资金流入渠道

### 2.1 散户资金

| 项目 | 说明 |
|------|------|
| **指标名称** | A股月度新开户数 |
| **数据存储** | `market_monitor/data/new_accounts.csv` |
| **数据格式** | period(YYYY-MM), new_accounts(万户), mom_pct(%), yoy_pct(%), source |
| **数据来源** | [上交所](http://www.sse.com.cn/services/tradestats/monthly/) |
| **获取方式** | 自动爬取 + CSV本地缓存 + 手动录入回退 |

#### 信号判断规则

| 新开户数水平 | 信号描述 |
|--------------|----------|
| 月度新开户数 > 300万户 | 🔴 情绪过热 |
| 月度新开户数 150-300万户 | 🟡 情绪偏热 |
| 月度新开户数 < 150万户 | 🟢 情绪未过热 |

#### 数据代码位置
- 数据获取：`market_monitor/data_sources/capital.py` → `fetch_new_accounts()`

---

### 2.2 杠杆资金

| 项目 | 说明 |
|------|------|
| **指标名称** | 两融余额、两融余额占流通市值、两融交易额占比 |
| **数据存储** | `market_monitor/data/margin.csv` |
| **数据格式** | 见下方详细字段说明 |
| **数据来源** | [东方财富融资融券](https://data.eastmoney.com/rzrq/) |
| **获取方式** | 东方财富 datacenter-web API 实时获取 |

#### 数据字段说明

| 字段名 | 含义 | 单位 |
|--------|------|------|
| `date` | 数据日期 | YYYY-MM-DD |
| `total_bal` | 两融余额 | 亿元 |
| `bal_chg` | 两融余额较前日变动 | 亿元 |
| `bal_chg_pct` | 两融余额较前日变动幅度 | % |
| `rz_mktcap_ratio` | 融资余额/A股流通市值 | % |
| `bal_mktcap_ratio` | 两融余额/A股流通市值 | % |
| `turnover_ratio` | 两融交易/全市场成交额 | % |

#### 信号判断规则

| 指标 | 阈值条件 | 信号描述 |
|------|----------|----------|
| 两融余额占流通市值 | > 2.5% | 🔴 过热 |
| 两融余额占流通市值 | 2.0% - 2.5% | 🟡 偏热 |
| 两融余额占流通市值 | < 2.0% | 🟢 中性 |
| 两融交易额占比 | > 10% | 🔴 过热 |
| 两融交易额占比 | 8% - 10% | 🟡 偏热 |
| 两融交易额占比 | < 8% | 🟢 中性 |

#### 数据代码位置
- 主接口：`market_monitor/data_sources/capital.py` → `RPTA_WEB_MARGIN_DAILYTRADE`
- 补充接口：`market_monitor/data_sources/capital.py` → `RPTA_RZRQ_LSHJ`
- 数据获取：`market_monitor/data_sources/capital.py` → `fetch_margin_data()`
