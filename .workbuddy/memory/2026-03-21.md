### 飞书市场监控报告 - 基本面板块格式调整

**需求**: 调整基本面板块展示格式，参考资金面分层缩进式展示

**修改内容**:

1. **feishu.py 基本面展示格式调整**
   - 第一行: 评分 + 总结 (已有的 summary)
   - 第二行: 万得全A（除金融、石油石化）PE及百分位
   - 第三个板块: 经济总量/结构 (GDP + 人均收入 + 三产占比)
   - 第四个板块: 宏观供需关系 (CPI + PPI + PPI-CPI剪刀差 + PMI)
   - 第五个板块: 宏观流动性 (M2 + 社融 + 10年国债)

2. **解读逻辑增强**
   - GDP: ≥5% 复苏良好，<3% 增长乏力
   - 三产占比: 同比提升→消费改善，下降→内需偏弱
   - 人均收入: ≥6% 强劲，4-6% 正常，<2% 承压
   - CPI: 0-3% 温和通胀，>3% 通胀压力，<0% 通缩
   - PPI: >0% 上行改善，<-2% 明显通缩
   - PPI-CPI剪刀差: >+2pp 上游优，<-2pp 下游承压
   - PMI: ≥50 扩张，45-50 临界，<45 收缩
   - M2: ≥10% 宽松，<6% 偏紧
   - 社融: ≥12% 旺盛，8-12% 正常，<5% 低迷
   - 10年国债: <2% 低利率利好成长，>3% 高估值承压

**数据源保持不变**:
- GDP: 东方财富 RPT_ECONOMY_GDP
- 人均收入: 国家统计局 A050102
- CPI/PPI/PMI: 东方财富三路接口
- M2: 东方财富 RPT_ECONOMY_CURRENCY_SUPPLY
- 社融: 国家统计局 A0L0801
- 10年国债: push2his K线接口
- 万得全A估值: Wind API
