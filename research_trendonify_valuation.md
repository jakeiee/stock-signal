# Trendonify 全球主要市场估值调研报告

**调研日期**: 2026-04-18

## 一、Trendonify 网站概况

### 1.1 网站基本信息

| 项目 | 内容 |
|------|------|
| **网站名称** | Trendonify |
| **网站地址** | https://trendonify.com |
| **核心功能** | 全球股市估值数据分析、PE比率对比 |
| **目标页面** | https://trendonify.com/pe-ratio/major-countries |
| **数据覆盖** | 美股、港股、日股、韩股等主要市场 |

### 1.2 访问限制

**严重问题**: Trendonify 网站部署了 **Cloudflare 安全验证**，阻止了自动化访问。

```
访问状态: 被拦截
拦截机制: Cloudflare Turnstile + 人机验证
错误信息: "Just a moment... Enable JavaScript and cookies to continue"
```

**尝试过的方法**:
- ❌ 直接 HTTP 请求: 被 Cloudflare 拦截
- ❌ Playwright 浏览器自动化: 卡在验证页面
- ❌ Web 抓取工具: 无法绕过 JavaScript 验证

## 二、替代数据源调研

由于 Trendonify 无法直接访问，调研了以下替代数据源：

### 2.1 乐咕乐股 (legulegu.com)

| 市场 | 页面URL | API状态 | 网页数据 |
|------|---------|---------|----------|
| **美股 (标普500)** | /stockdata/market/sandp | ❌ API 404 | ✅ PE=30.62 (2026-04-15) |
| **港股 (恒生指数)** | /stockdata/market/hsi | ❌ API 404 | ✅ PE=13.52 (2026-03-31) |
| **日股 (日经225)** | /stockdata/market/nikkei | ❌ 页面 404 | ❌ 无数据 |
| **韩股 (KOSPI)** | /stockdata/market/kospi | ❌ 页面 404 | ❌ 无数据 |

**结论**: 乐咕乐股仅支持美股和港股的PE数据，日股和韩股数据缺失。

### 2.2 MacroView (macroview.club)

| 项目 | 内容 |
|------|------|
| **页面** | https://www.macroview.club/charts?name=global_pe |
| **数据日期** | 2026-04-17 |
| **上证指数PE** | 17.12 |
| **其他市场** | 通过 JavaScript 动态加载，无法直接抓取 |

**结论**: 数据通过 JS 动态加载，无法通过简单 HTTP 请求获取。

### 2.3 东方财富 (data.eastmoney.com)

| 项目 | 内容 |
|------|------|
| **页面** | https://data.eastmoney.com/gzfx/ |
| **数据覆盖** | 主要覆盖 A 股市场 |
| **全球估值** | ❌ 不直接提供美股/港股/日股/韩股PE对比 |

**结论**: 东方财富主要服务 A 股市场，全球估值数据有限。
